# 🛟 SafePal Pro — Disaster Preparedness App (Pakistan)

• One-tap SOS • GPS location**

SafePal Pro is a comprehensive disaster preparedness application designed for Pakistan's emergency response system. It provides instant access to emergency services, first aid guides, evacuation procedures, and personal emergency contacts.

## 🚨 Emergency Numbers (Pakistan)

| Service | Number |
|---------|--------|
| 🚓 Police | **15** |
| 🚑 Ambulance | **115** |
| 🔥 Fire Brigade | **16** |

## ✨ Features

### 📞 Emergency Contacts
- One-tap call to Police (15), Ambulance (115), and Fire Brigade (16)
- Save unlimited personal emergency contacts
- One-tap call and SMS to any saved contact

### 🆘 SOS System
- Big red SOS button for emergencies
- Automatically includes your saved location in SOS message
- Sends SOS via SMS to all saved emergency contacts
- Customizable SOS message template

### 🗺️ Evacuation Guides (7 Disasters)
- Fire Evacuation
- Earthquake Safety
- Flood Response
- Hurricane Preparedness
- Tsunami Response
- Tornado Safety
- Landslide Safety

### 🩺 First Aid Guides (10 Emergencies)
- Burns Treatment
- Cuts & Bleeding
- CPR (Cardiopulmonary Resuscitation)
- Fracture / Broken Bone
- Choking (Heimlich Maneuver)
- Heart Attack Signs
- Stroke (FAST Method)
- Seizure First Aid
- Snake Bite Response
- Hypothermia Treatment

### 🎒 Emergency Checklist
- 15 essential emergency kit items
- Progress bar shows preparedness percentage
- Saves your progress automatically
- Reset checklist option

### 📍 Location Tools
- Get current GPS coordinates (works offline)
- Save your address / building name / landmarks
- Share your location via SMS to emergency contacts
- Location auto-inserts into SOS messages using `[LOCATION]` tag

### ⚙️ Settings & Data Management
- 💾 Export all data (contacts, checklist, location, SOS template) as JSON backup
- 📥 Import previously saved backup
- 🗑️ Reset all data

### 📱 Installable as PWA
- Add to home screen like a real app
- Works completely offline after first visit
- Custom app icon and splash screen

## 🎯 How to Use

1. **Open the app** (works immediately, no internet needed)
2. **Add your emergency contacts** (family, friends, neighbors)
3. **Save your location** (address or GPS coordinates)
4. **Customize your SOS message** template
5. **Prepare your emergency kit** using the checklist
6. **Review evacuation and first aid guides** before emergencies
7. **In an emergency, tap the red SOS button** — it will open SMS for all your contacts with your location

## 📦 Installation

### As a Web App (PWA)
1. Open the live URL in Chrome/Safari
2. Tap the share menu
3. Select "Add to Home Screen"
4. The app will install with a custom icon

### As Source Code
1. Clone or download the repository
2. Open `index.html` in any browser
3. Works immediately (no build step needed)
